Objective 10: Grid Acceleration

Implemented grid acceleration by spliting the world into n by n by n cubes, and intersect with only cubes that the ray intersects with. From the chart, as the size of the cube increases, more objects are contained within it, therefore the time to render the scene increases.

Extra Objective 1: Multi-threaded Raytracing

Split image into n by n squares and raytraced each square concurrently. From the chart, as the number of thread increases, time taken to render the scene decreases, however, speed up become less and less noticable as number of thread increases.
